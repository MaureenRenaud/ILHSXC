# ILHSXC
A look at performance related factors in Illinois High School Cross-Country


This project began as an assignment for STAT 485, a class I took while pursuing my Master's Degree in Applied Statistics.  I have included the original draft that I submitted and I plan to include reworked versions as I improve my skills in R and in Statistics.
The text files contain the original data from each year that the race was held.  I combined them all into an excel document that I read into R.
